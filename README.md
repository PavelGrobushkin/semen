# SEMEN — SEgmentation for MEthylation Noise reduction

This git repository contains code that was in the SEMEN project. Sparsity of the bulk and (especially) single-cell DNA methylation data is a major challenge that influences performance of models based on these data (such as age prediction models). In the project, we aimed to enhance the tracing of age- and disease-related epigenetic changes using segmented DNA methylation data.

## Overview

In SEMEN, we cluster CpG sites into methylation segments and use them as features for age prediction, aiming to reduce noise and improve interpretability compared to site-level models.

## Reproducibility 

To set up the environment used in the project, run the following commands:

``` bash
mamba env create -f environment.yml
mamba activate semen
```
